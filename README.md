# GovFlow Landing Page

Standalone implementation of the GovFlow marketing/landing page, extracted from the main BASE44-generated application.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

You can deploy the `dist` folder to any static host (Vercel, Netlify, etc.).
